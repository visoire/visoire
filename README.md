## How I Approach Projects

When it's a startup or new product, I go with Next.js, TypeScript, and PostgreSQL. Clean, modern, scalable. When it's a traditional business website or you need to keep costs down, PHP with MySQL works perfectly fine. Both get the job done right.

Databases get designed from the start with your actual needs in mind. I write queries that don't slow down. I configure servers so they handle load without crashing.

Security matters. I implement OAuth2 when you need it, use JWT for APIs, and don't leave holes open.
